# Independent-church-website
This website takes the details of people  in the church .It assigns each member a district he/she belongs in according to the details they have provided.It also shows an individual which level they are in the church i.e Elder,Youth,Teenager and child level.Finally ,it also shows people who are disabled in our church.

# Christ Saves church Website

This is a simple church registration web application built with CSS,HTML and Javascript.

## Overview
This website collects member details and assigns them to a church district and category.

#Features

* Navigation bar

* Mission &Vision section
         
* Member registration form

* Area selection (Donholm, Pipeline, Nyayo)

* Gender & physical status selection

* Basic form validation with JavaScript

* District and category assignment


# Technologies used
* HTML
* CSS
* Javascript
